## PC Commend
- “C”	: Clock_Time
- “W”	: Stop_Watch_Time
- “T” : Timer_Time
- “U” : Ultrasensor_Value
- “D”	: DHT11_Values
- "S" : Setting Mode
- "u" : Up / Timer Off
- "d" : Down
- "l" : left move / Stop_Watch, Timer Clear
- "r" : right move / Stop_Watch, Timer Run/Stop

## FPGA Butten Message
- Btn_U : "Up"
- Btn_D : "Down"
- Btn_L : "Left", "Clear"
- Btn_R : "Right", "Run", "Stop"
